makarajothi-portfolio

Files:
- index.html  : main portfolio page
- style.css   : styles for the page

How to preview locally:
1. Extract the zip to a folder.
2. Open index.html in your browser (double-click) OR use VS Code Live Server extension:
   - Open the folder in VS Code
   - Right-click index.html -> Open with Live Server

How to upload to GitHub:
1. Create a new repository on GitHub (e.g., makarajothi-portfolio).
2. In your project folder run:
   git init
   git add .
   git commit -m "Initial commit - portfolio"
   git branch -M main
   git remote add origin https://github.com/yourusername/makarajothi-portfolio.git
   git push -u origin main
3. Enable GitHub Pages in repo Settings -> Pages -> Deploy from branch -> main -> Save.
4. Your site will be live at: https://yourusername.github.io/makarajothi-portfolio/
